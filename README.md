# Personal Portfolio Website

## Project Overview

This project is a simple Personal Portfolio Website created using HTML. The website contains information about me, my skills, hobbies, profile picture, and contact details.

## HTML Concepts Learned

* HTML Document Structure
* Headings and Paragraphs
* Lists
* Images and Alt Text
* Hyperlinks
* Internal Navigation Links
* Forms and Input Validation

## Setup Instructions

1. Download the project files.
2. Open the project folder in Visual Studio Code.
3. Open the `index.html` file.
4. Run the file using Live Server or open it directly in a web browser.
